# Getting Started with JerryBB

JerryBB is an assistant manager, scout, and analyst for BuzzerBeater. It brings together your team data, opponent preparation, tactical history, player development, and match analysis so you can make better decisions before and after games.

## What JerryBB is for

Use JerryBB when you want a clearer pre-game plan or a better post-game read on what happened. It is built for managers who already have enough BuzzerBeater context to care about tactics, sample size, and matchup detail.

## The main product loop

JerryBB has two connected loops:

- Before a match: open **Home**, review the next opponent, then open that opponent's **Briefing**.
- If the briefing feels thin or you want more context, open **Opponent Tactical History** to see whether the same behavior shows up across multiple games.
- After a match: use **Match Analysis** for the completed game, then return to **My Team Tactical History** or the opponent's history when you want to compare patterns over time.

The briefing is the immediate recommendation layer. Tactical history is the longer view that helps you decide whether the recommendation is backed by a repeatable pattern or just the latest data point.

## Prerequisites and data readiness

JerryBB is most useful when the underlying data is current enough to support the page you opened.

- Sign in to the workspace first.
- Sync your BuzzerBeater account data so Home and roster pages have current team and schedule context.
- Import boxscores and play-by-play when you want tactical history, shot-zone evidence, or deeper pattern analysis.
- Treat empty states, roster-only briefings, and low-sample outputs as “need more data,” not as final conclusions.

The implementation currently falls back to a reduced briefing when only roster data is available, and tactical-history pages stay empty until enough boxscore or PBP evidence has been imported.

## Guided first session

1. Open **Home** and check the next opponent, the time to tip, and any freshness or import indicators.
2. Open the opponent's **Briefing** and read the recommendation first.
3. If the recommendation feels uncertain, switch to **Opponent Tactical History** and compare the sample size, trend, and recent evidence.
4. If the game has already been played, use **Match Analysis** for the single-game readout.
5. Come back to **My Team Tactical History** when you want to compare your own tactical patterns across multiple games.

## Briefing versus historical analysis

These pages answer different questions:

- **Briefing:** What should I consider for this upcoming game?
- **Opponent Tactical History:** What patterns have appeared across previous games?
- **My Team Tactical History:** What patterns have appeared in my own games?

Use the briefing for the immediate matchup. Use historical pages to test whether a pattern is recurring, how much data supports it, and whether the current recommendation needs more caution.

## Limitations and confidence

- Historical evidence is only as strong as the imported games behind it.
- Tactical history can be empty, partial, or low-confidence when boxscores and PBP captures are missing.
- A strong-looking result in one game does not prove the tactic caused the outcome.
- JerryBB is not a live-game companion; it is a pre-game and post-game analysis tool.
